Política de Privacidade

ControlAGRO — Gestão de Visitas e Clientes

Última atualização: 17 de março de 2026

1 Identificação do Responsável

Esta política de privacidade descreve como o aplicativo ControlAGRO, de titularidade de O Fazendeiro, inscrita no CNPJ sob o número 21.899.992/0001-00, com sede em Sete Lagoas, MG, trata os dados pessoais dos usuários e de terceiros cadastrados.

Para questões relacionadas a esta política, entre em contato pelo email: marina.ofazendeiro@gmail.com

2 Dados Coletados

O ControlAGRO coleta e processa os seguintes tipos de dados:

2.1. Dados de cadastro dos vendedores

  • Nome completo
  • Endereço de email
  • Telefone

2.2. Dados de clientes registrados

  • Nome completo ou razão social
  • CPF ou CNPJ
  • Telefone
  • Endereço de email
  • Endereço da propriedade rural
  • Cidade e estado
  • Nome da propriedade

2.3. Dados de localização

  • Coordenadas GPS (latitude e longitude) registradas no momento das visitas técnicas

2.4. Fotografias

  • Imagens capturadas pela câmera do dispositivo ou selecionadas da galeria durante as visitas técnicas

2.5. Dados comerciais

  • Informações sobre culturas e plantios dos clientes
  • Valores estimados de vendas
  • Motivos e descrições das visitas
  • Status de negociações
  • Registros de contatos e acompanhamento

3 Finalidade da Coleta

Os dados são coletados e tratados para as seguintes finalidades:

  • Gestão da equipe de vendas e acompanhamento de visitas técnicas
  • Registro e acompanhamento de clientes do setor agrícola
  • Geolocalização para comprovação de visitas realizadas em campo
  • Geração de relatórios gerenciais internos
  • Funcionamento offline do aplicativo com sincronização posterior

4 Base Legal (LGPD)

O tratamento dos dados pessoais pelo ControlAGRO fundamenta-se nas seguintes bases legais previstas na Lei Geral de Proteção de Dados (Lei nº 13.709/2018):

  • Execução de contrato ou procedimentos preliminares (Art. 7º, V): os dados são necessários para a execução das atividades de gestão de vendas e visitas técnicas
  • Legítimo interesse do controlador (Art. 7º, IX): para fins de relatórios gerenciais, melhoria do serviço e comprovação de visitas realizadas
  • Consentimento do titular (Art. 7º, I): quando aplicável, o consentimento é obtido de forma explícita no primeiro acesso ao aplicativo

5 Compartilhamento de Dados

  • Supabase: infraestrutura de banco de dados e autenticação (provedor cloud hospedado na AWS)
  • Gestor da equipe: o gestor tem acesso aos dados de visitas e clientes registrados pelos vendedores da equipe

Os dados pessoais não são vendidos, alugados ou compartilhados com terceiros para fins de marketing, publicidade ou qualquer outra finalidade comercial não relacionada à operação do aplicativo.

6 Armazenamento e Segurança

  • Os dados são armazenados em servidores Supabase (PostgreSQL), com criptografia em trânsito (TLS) e em repouso
  • O aplicativo utiliza armazenamento local no dispositivo via IndexedDB para funcionamento offline. Os dados ficam cacheados localmente até a sincronização com o servidor
  • A autenticação é realizada via Supabase Auth com tokens JWT
  • O banco de dados utiliza Row Level Security (RLS), garantindo que cada vendedor acesse apenas seus próprios dados
  • As fotos são armazenadas em storage seguro com controle de acesso

7 Retenção de Dados

  • Os dados pessoais são mantidos enquanto o vendedor estiver ativo no sistema
  • Após o desligamento, os dados podem ser mantidos pelo período necessário para cumprimento de obrigações legais ou regulatórias
  • A exclusão dos dados pode ser solicitada ao gestor da equipe ou pelo email de contato

8 Direitos do Titular (LGPD Art. 18)

Em conformidade com a LGPD, o titular dos dados pessoais tem direito a:

  • Confirmação da existência de tratamento de dados
  • Acesso aos dados pessoais coletados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários, excessivos ou tratados em desconformidade
  • Portabilidade dos dados a outro fornecedor de serviço
  • Eliminação dos dados tratados com consentimento
  • Informação sobre entidades públicas e privadas com as quais houve compartilhamento
  • Informação sobre a possibilidade de não fornecer consentimento e as consequências
  • Revogação do consentimento a qualquer momento

9 Permissões do Aplicativo

O ControlAGRO solicita as seguintes permissões no dispositivo:

  • Câmera: para registrar fotos das visitas técnicas
  • Localização (GPS): para registrar as coordenadas geográficas das visitas realizadas
  • Galeria de fotos: para selecionar imagens existentes no dispositivo
  • Internet: para sincronização dos dados com o servidor
  • Armazenamento local: para funcionamento offline do aplicativo

Todas as permissões são solicitadas no momento do uso e podem ser revogadas a qualquer momento nas configurações do dispositivo.
