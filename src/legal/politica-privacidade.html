Politica de Privacidade

ControlAGRO — Aplicativo de Gestao de Visitas e Clientes

Ultima atualizacao: 17 de marco de 2026

1. Identificacao do Responsavel

Para questoes relacionadas a esta politica, entre em contato pelo email: marina.ofazendeiro@gmail.com

2. Dados Coletados

O ControlAGRO coleta e processa os seguintes tipos de dados:

2.1. Dados de cadastro dos vendedores

  • Nome completo
  • Endereco de email
  • Telefone

2.2. Dados de clientes registrados

  • Nome completo ou razao social
  • CPF ou CNPJ
  • Telefone
  • Endereco de email
  • Endereco da propriedade rural
  • Cidade e estado
  • Nome da propriedade

2.3. Dados de localizacao

  • Coordenadas GPS (latitude e longitude) registradas no momento das visitas tecnicas

2.4. Fotografias

  • Imagens capturadas pela camera do dispositivo ou selecionadas da galeria durante as visitas tecnicas

2.5. Dados comerciais

  • Informacoes sobre culturas e plantios dos clientes
  • Valores estimados de vendas
  • Motivos e descricoes das visitas
  • Status de negociacoes
  • Registros de contatos e acompanhamento

3. Finalidade da Coleta

Os dados sao coletados e tratados para as seguintes finalidades:

  • Gestao da equipe de vendas e acompanhamento de visitas tecnicas
  • Registro e acompanhamento de clientes do setor agricola
  • Geolocalizacao para comprovacao de visitas realizadas em campo
  • Geracao de relatorios gerenciais internos
  • Funcionamento offline do aplicativo com sincronizacao posterior

4. Base Legal (LGPD)

O tratamento dos dados pessoais pelo ControlAGRO fundamenta-se nas seguintes bases legais previstas na Lei Geral de Protecao de Dados (Lei n. 13.709/2018):

  • Execucao de contrato ou procedimentos preliminares (Art. 7, V): os dados sao necessarios para a execucao das atividades de gestao de vendas e visitas tecnicas
  • Legitimo interesse do controlador (Art. 7, IX): para fins de relatorios gerenciais, melhoria do servico e comprovacao de visitas realizadas
  • Consentimento do titular (Art. 7, I): quando aplicavel, o consentimento e obtido de forma explicita no primeiro acesso ao aplicativo

5. Compartilhamento de Dados

  • Supabase: infraestrutura de banco de dados e autenticacao (provedor cloud hospedado na AWS)
  • Gestor da equipe: o gestor tem acesso aos dados de visitas e clientes registrados pelos vendedores da equipe

Os dados pessoais nao sao vendidos, alugados ou compartilhados com terceiros para fins de marketing, publicidade ou qualquer outra finalidade comercial nao relacionada a operacao do aplicativo.

6. Armazenamento e Seguranca

  • Os dados sao armazenados em servidores Supabase (PostgreSQL), com criptografia em transito (TLS) e em repouso
  • O aplicativo utiliza armazenamento local no dispositivo via IndexedDB para funcionamento offline. Os dados ficam cacheados localmente ate a sincronizacao com o servidor
  • A autenticacao e realizada via Supabase Auth com tokens JWT
  • O banco de dados utiliza Row Level Security (RLS), garantindo que cada vendedor acesse apenas seus proprios dados
  • As fotos sao armazenadas em storage seguro com controle de acesso

7. Retencao de Dados

  • Os dados pessoais sao mantidos enquanto o vendedor estiver ativo no sistema
  • Apos o desligamento, os dados podem ser mantidos pelo periodo necessario para cumprimento de obrigacoes legais ou regulatorias
  • A exclusao dos dados pode ser solicitada ao gestor da equipe ou pelo email de contato

8. Direitos do Titular (LGPD Art. 18)

Em conformidade com a LGPD, o titular dos dados pessoais tem direito a:

  • Confirmacao da existencia de tratamento de dados
  • Acesso aos dados pessoais coletados
  • Correcao de dados incompletos, inexatos ou desatualizados
  • Anonimizacao, bloqueio ou eliminacao de dados desnecessarios, excessivos ou tratados em desconformidade
  • Portabilidade dos dados a outro fornecedor de servico
  • Eliminacao dos dados tratados com consentimento
  • Informacao sobre entidades publicas e privadas com as quais houve compartilhamento
  • Informacao sobre a possibilidade de nao fornecer consentimento e as consequencias
  • Revogacao do consentimento a qualquer momento

9. Permissoes do Aplicativo

O ControlAGRO solicita as seguintes permissoes no dispositivo:

  • Camera: para registrar fotos das visitas tecnicas
  • Localizacao (GPS): para registrar as coordenadas geograficas das visitas realizadas
  • Galeria de fotos: para selecionar imagens existentes no dispositivo
  • Internet: para sincronizacao dos dados com o servidor
  • Armazenamento local: para funcionamento offline do aplicativo

Todas as permissoes sao solicitadas no momento do uso e podem ser revogadas a qualquer momento nas configuracoes do dispositivo.
